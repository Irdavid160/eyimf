How to add your media (quick)

  1. For YouTube videos, use the embed URL: https://www.youtube.com/embed/VIDEO_ID inside the iframe src.
  2. Upload audio files to assets/media/audio/ (mp3 recommended) and update the audio src path.
  3. Upload images to assets/media/images/ and update the img src paths and alt text.
  4. Commit changes and then clear browser cache or bump the CSS/JSON cache-bust query strings if you don't see updates.
